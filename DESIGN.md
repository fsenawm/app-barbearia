# DESIGN.md - App Barbearia

## 1. Categoria Visual
Dashboard Administrativo Mobile-First / Modern Dark Mode.

## 2. Cores
- **Fundo:** Dark (#000000 a #121212)
- **Primária:** Blue (#177ccf)
- **Texto:** White (#FFFFFF), Gray (#9ca3af)
- **Sucesso:** Green (#10b981)
- **Erro/Alerta:** Red (#ef4444)

## 3. Tipografia
- **Fonte:** Manrope (Google Fonts)
- **Tamanhos:** Base 16px, Títulos 24px-32px.

## 4. Componentes Base
- **Cards:** Bordas arredondadas (`rounded-lg` ou `rounded-2xl`), fundo levemente cinza/preto.
- **Botões:** `bg-[#177ccf] text-white rounded-md px-4 py-2 hover:opacity-90 transition-all`.
- **Inputs:** `bg-transparent border border-gray-700 rounded-md focus:border-[#177ccf]`.
